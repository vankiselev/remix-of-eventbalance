

## План: Автоматическое сжатие и круговой кроппер для аватара

### Что будет сделано

1. **Утилита для сжатия изображений** - клиентское сжатие через Canvas API
2. **Компонент круглого кроппера** - интерфейс для позиционирования фото в кружке
3. **Интеграция в ProfilePage** - замена простой загрузки на кроппер

### Детали реализации

#### 1. Утилита сжатия изображений

**Файл:** `src/utils/imageCompression.ts`

Функционал:
- Сжимает изображение до максимального размера (например, 1200x1200 для аватаров)
- Конвертирует в JPEG с качеством 0.85 (хороший баланс размера и качества)
- Работает через Canvas API (без внешних библиотек)
- Автоматически уменьшает качество если файл все еще большой

```text
Алгоритм сжатия:
┌─────────────────────────────────┐
│  Загрузка файла (любой размер) │
└─────────────────┬───────────────┘
                  │
                  ▼
┌─────────────────────────────────┐
│  Создание Image element        │
└─────────────────┬───────────────┘
                  │
                  ▼
┌─────────────────────────────────┐
│  Расчет новых размеров         │
│  (max 1200x1200, сохраняя      │
│   пропорции)                   │
└─────────────────┬───────────────┘
                  │
                  ▼
┌─────────────────────────────────┐
│  Отрисовка на Canvas           │
└─────────────────┬───────────────┘
                  │
                  ▼
┌─────────────────────────────────┐
│  Конвертация в Blob            │
│  (JPEG, качество 0.85)         │
└─────────────────┬───────────────┘
                  │
                  ▼
┌─────────────────────────────────┐
│  Если > 5MB: уменьшить качество│
│  до 0.7, затем 0.5             │
└─────────────────┴───────────────┘
```

#### 2. Компонент круглого кроппера

**Файл:** `src/components/ui/avatar-cropper.tsx`

Функционал:
- Модальное окно (Dialog)
- Круглая маска для предпросмотра
- Управление:
  - **Zoom (приближение)** - слайдер от 1x до 3x
  - **Pan (перемещение)** - перетаскивание мышью/пальцем
  - **Rotate (поворот)** - слайдер от -180° до +180°
- Кнопки "Сохранить" и "Отмена"

Интерфейс:
```text
┌────────────────────────────────────────┐
│           Настройка фото               │
├────────────────────────────────────────┤
│                                        │
│    ┌──────────────────────────┐        │
│    │                          │        │
│    │     ○─────────────○      │        │
│    │    /               \     │        │
│    │   │    [ФОТО]       │    │        │
│    │    \               /     │        │
│    │     ○─────────────○      │        │
│    │       (круглая маска)    │        │
│    └──────────────────────────┘        │
│                                        │
│  Масштаб:  [─────●───────────]  1.5x   │
│  Поворот:  [───────●─────────]  15°    │
│                                        │
├────────────────────────────────────────┤
│     [ Отмена ]        [ Сохранить ]    │
└────────────────────────────────────────┘
```

Технические детали:
- Используем Canvas для финального рендера обрезанного круга
- Touch events для мобильных устройств
- Плавные анимации при zoom/pan

#### 3. Обновление ProfilePage

**Файл:** `src/pages/ProfilePage.tsx`

Изменения:
1. Добавить состояние для диалога кроппера
2. При выборе файла → открыть кроппер вместо прямой загрузки
3. После подтверждения в кроппере → сжать и загрузить


### Преимущества

- **Без новых зависимостей** - используем Canvas API
- **Качественное сжатие** - JPEG с адаптивным качеством
- **Удобный UX** - пользователь видит результат до сохранения
- **Работает на мобильных** - touch события для перетаскивания

### Техническая информация

- Canvas API для сжатия и обрезки изображений
- toBlob() для конвертации canvas в файл
- CSS clip-path для круглой маски в превью
- Radix Dialog для модального окна
- Radix Slider для zoom/rotate
