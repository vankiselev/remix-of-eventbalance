
Цель: убрать пустую белую область внизу виджета «Ближайшие мероприятия», чтобы контент/скролл заполнял карточку до нижней границы, и обрезка происходила по внешней рамке самой карточки.

1) Найти и убрать фиксированную высоту списка
- В `TodayEventsCard` сейчас у `ScrollArea` стоит `h-[400px]`, из-за этого внутри высокой карточки (`h-full`, `row-span-2`) остаётся пустое место.
- Заменить на растягивание по доступной высоте: `flex-1 min-h-0` (внутри `CardContent` уже есть `flex`).

2) Подогнать внутренние отступы, чтобы не было визуального «обреза»
- Проверить/скорректировать отступы у контейнера списка (`space-y-3 pr-3 pb-*`).
- Оставить только минимальный нижний отступ (или убрать), чтобы последняя карточка доходила до низа естественно, без отдельной «белой подложки».

3) Сохранить обрезку по внешней рамке карточки
- Не менять внешнюю структуру `Card`/`CardContent`; текущий `overflow-hidden` у контейнера контента использовать как клип по основной рамке.
- Проверить, что скролл остаётся внутри виджета и не ломает радиусы/бордеры.

4) Быстрая проверка после правки
- Много мероприятий: список прокручивается, пустой зоны внизу нет.
- Мало мероприятий: контент выглядит аккуратно, без артефактов.
- На текущем десктопном размере (как на скрине) визуально нижний край совпадает с рамкой «Ближайшие мероприятия».

Технические детали (что именно изменю)
- Файл: `src/components/dashboard/TodayEventsCard.tsx`
- Основная правка классов:
  - `ScrollArea`: `h-[400px]` → `flex-1 min-h-0`
  - при необходимости тонкая правка `pb-1` у внутреннего контейнера списка, чтобы получить нужный визуальный край без пустой зоны.
